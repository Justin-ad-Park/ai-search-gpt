엘라스틱 서치로 AI 검색을 하는 토이프로젝트를 로컬 PC에서 개발하고 싶어.
AI 검색은 벡터 검색으로 한글 단어 간의 유사도를 기반으로 검색하는 방식을 사용하거나, 더 나은 AI 검색 방법이 있으면 이를 이용하고 싶어.

상용 서비스 비용을 고려해서 유료 라이선스를 가급적 사용하지 않고, 무료 라이선스 또는
아주 저렴한 유지 비용으로 서비스 가능하도록 구성해야 해

로컬 PC에는 도커 기반 쿠버네티스 서비스가 떠 있고, kubectl 명령어도 사용 가능하게 셋팅이 되어 있어.

1. ./shell 폴더에 엘라스틱 서치 설치 스크립트 및 구동, 구동 확인 스크립트 생성
   - 기존 설치 스크립트에 문제가 있으면 기존 설치된 엘라스틱 서치 삭제 및 스크립트 재생성 후 재설치

2. 식품 쇼핑몰 샘플 데이터를 약 100개~1000개 정도 임의로 만들 것
   - 유사도 기반 검색에 적합한 식품 상품 데이터 구성
   - 초기를 학습에 용이하도록 구성할 것. DB를 사용한다면 H2로 구성. 또는 파일(JSON)로 구성해도 됨

3. 엘라스틱 서치 AI 검색이 가능하도록 구성하되, 무료 라이선스로 구성할 것.
   - 평가판이 아닌 무료로 AI 검색 구성을 원함
   - 유료 라이선스가 필요하다면 비용이 저렴해야 함

4. AI 색인을 Spring boot java application으로 만듬

5. 색인이 끝난 후 Spring Boot의 테스트(Junit)를 통해 AI 유사도 검색이 잘 동작함을 검증
    - 테스트 시 검색 결과를 shut으로 출력
